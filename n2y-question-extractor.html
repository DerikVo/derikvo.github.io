Extract individual questions from PDF worksheets and enlarge them for better accessibility. This tool uses OCR technology to automatically detect questions and create one-question-per-page PDFs with adjustable sizing.

How to Use This Tool

  1. Upload PDF: Click the upload button and select your PDF worksheet
  2. Select Page: Choose which page contains the questions you want to extract
  3. Set Enlargement: Use the slider to adjust how much to enlarge the questions (1x-10x)
  4. Process: Click "Process PDF" and wait for the tool to extract questions
  5. Download: Download your new PDF with one enlarged question per page

Tips for Best Results

  • Use PDFs with clearly numbered questions (e.g., "1.", "2.", "Question 1")
  • Ensure questions are well-spaced on the original page
  • Start with a smaller enlargement factor (4x-6x) and adjust as needed
  • For students with visual impairments, try larger factors (8x-10x)
  • The tool works best with text-based PDFs, but also supports scanned documents via OCR
